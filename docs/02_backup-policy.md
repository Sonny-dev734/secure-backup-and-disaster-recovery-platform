# 02 - Backup Policy

## Purpose
Define rules for secure, reliable backup of critical data.

## Scope
This policy applies to all personal and project data on the workstation.

## Backup Rules (3-2-1)
- 3 copies of data
- 2 different media types
- 1 copy offsite (will be added later)

## Retention Policy
- Daily backups: kept 7 days
- Weekly backups: kept 4 weeks  
- Monthly backups: kept 12 months

## Encryption Standard
- AES-256 (Restic)
- Password stored in restricted file (600 permissions)
